# ♻️ GREENSITE - Smart Recycling Platform

A modern, interactive recycling platform website built with HTML, CSS, and JavaScript.

## Features

- **📊 Live Pricing**: Real-time recycling material prices with trend indicators
- **🧮 Recycle Calculator**: Calculate potential earnings from recycled materials
- **🗺️ Location Finder**: Find nearby recycling centers using your location and OpenStreetMap
- **🌍 Educational Gallery**: Environmental awareness posters and tips
- **🏆 Weekly Challenges**: Engage users with recycling challenges
- **📱 Responsive Design**: Works on desktop, tablet, and mobile devices

## How to Use

1. Visit the live site: [GREENSITE on GitHub Pages](https://rith063.github.io/GREENSITE/)
2. Navigate through sections using the top menu
3. Check current recycling prices with live trends
4. Calculate potential earnings using the calculator
5. Find nearby recycling centers
6. Learn about recycling tips and environmental facts

## Technologies Used

- **HTML5** - Structure
- **CSS3** - Styling with gradients, animations, and responsive design
- **JavaScript** - Interactivity and dynamic features
- **Leaflet.js** - Interactive mapping
- **OpenStreetMap** - Map data
- **Overpass API** - Recycling center data

## Key Features Explained


### Calculator
Select a material, enter quantity, and get instant earnings calculation with motivational messages.

### Location Services
Uses browser geolocation to find your position and displays nearby recycling centers on a map with navigation options.

### Educational Content
- Recycling tips in Malay
- Environmental facts
- Weekly challenges to encourage participation

## Installation

No installation required! Simply open `index.html` in a web browser or visit the hosted version above.
